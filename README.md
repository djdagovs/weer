Weer in Koudekerke
====

A real-time weather information website driven by a Personal Weather Station (Davis Vantage Pro2) and a Raspberry Pi. The data is retrieved from the weather station by [WeeWX](https://github.com/weewx/weewx). Forecast information is provided by Wunderground. The [WeeWX Pusher extension](https://github.com/raymondelooff/weewx-pusher) is being used to push real-time weather data to the website.
